# Security Reports

This directory contains per-platform security artifacts generated by the CI/CD pipeline.

## Structure

```
security/
  shared/          # Cross-platform: npm dependencies, filesystem scan
    dashboard.html # HTML vulnerability + dependency dashboard
    sbom.cdx.json  # CycloneDX SBOM
    vulnerabilities.json
  macos/           # macOS-specific: includes Cargo/Rust deps
    dashboard.html
    sbom.cdx.json
    vulnerabilities.json
  windows/         # Windows-specific
    dashboard.html
    sbom.cdx.json
    vulnerabilities.json
  linux/           # Linux-specific
    dashboard.html
    sbom.cdx.json
    vulnerabilities.json
```

## How it works

1. On every release tag (`v*`), the pipeline runs:
   - Unit tests (102 tests)
   - npm audit (SCA for JavaScript)
   - cargo audit (SCA for Rust)
   - TruffleHog (secrets detection)
   - Trivy filesystem scan (vulnerability detection)
   - Trivy SBOM generation (CycloneDX format)

2. Per-platform dashboards are generated showing:
   - Vulnerability count by severity (Critical/High/Medium/Low)
   - Full dependency inventory with versions and ecosystems
   - Links to raw SBOM and vulnerability JSON

3. All reports are bundled into `security-reports.zip` and attached to the GitHub Release.

## Local generation

```bash
# Install trivy: https://aquasecurity.github.io/trivy/
trivy fs --format cyclonedx --output sbom.cdx.json .
trivy fs --format json --output vulns.json .
bash scripts/generate-security-dashboard.sh local sbom.cdx.json vulns.json security/local
open security/local/dashboard.html
```
