# Decisions — 2026-03-22 (Session 2)

## D1: Design System — "Soft Modern" inspired by market leaders

**Decision:** Use a soft, modern UI with neutral warm palette, subtle shadows, large rounded corners (12-16px), and glassmorphism-lite effects. Inspired by Things 3, Linear, and Apple Reminders.

**Alternatives considered:**
- Material Design 3: Too Google-specific, harder to differentiate
- Neumorphism: Accessibility concerns, poor contrast
- Sharp/angular: User explicitly said "soft"

**Why:** User wants "best in market" and "soft UI" with flexibility to change. A design-token-based approach with Tailwind makes the entire theme swappable.

---

## D2: Hosting Strategy — Local-first, portable

**Decision:** Docker Compose for local dev (Postgres + Redis + API). Web deploys to Vercel free tier. API containerized for any host (Railway, Fly.io, VPS, etc).

**Why:** User wants low cost and portability. Docker ensures no vendor lock-in. Vercel Hobby tier is free with generous limits.

---

## D3: Auth — Passport.js with JWT + OAuth providers

**Decision:** Email/password auth with bcrypt + JWT/refresh tokens. Google and Apple OAuth via Passport strategies. OAuth providers configured via env vars so they can be enabled/disabled.

**Why:** User chose Option B. OAuth from Day 1 means the auth module is built once correctly.

---

## D4: Notification Urgency — User-selectable per reminder

**Decision:** Add `priority` field to reminders: `critical` (persistent, repeat), `normal` (standard push), `low` (silent/badge only). User sets per reminder.

**Why:** User explicitly requested user-selectable urgency levels.

---

## D5: Security Pipeline — SAST + dependency scanning + threat model

**Decision:** GitHub Actions CI includes: ESLint security plugin, npm audit, CodeQL, Trivy container scanning. STRIDE threat model documented in `docs/THREAT_MODEL.md`.

**Why:** User explicitly requested vulnerability checking and threat modeling in the pipeline.

---

## D6: App Name Abstraction

**Decision:** App name stored in a single config constant (`APP_NAME`). All references use this constant. Makes renaming a one-line change.

**Why:** User confirmed "RemindMe" is temporary — prep for change.

---

## D7: Desktop for Testing Only

**Decision:** Build Electron app that compiles to unsigned .dmg for macOS. No code signing, no auto-update, no store distribution. Just enough to test locally.

**Why:** User wants to test on Mac now. Signing and distribution are Phase 2.
